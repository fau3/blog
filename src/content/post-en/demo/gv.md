Once everything is ready, you can start the registration. There is one more point here: your Google account matters too.

**The Google account must be associated with the United States in order to enter GV.**

**If a newly registered Gmail account was created by skipping phone verification, and you immediately use it to register GV, the success rate is high.**

**By contrast, a brand-new Gmail account created through an SMS code receiving service usually cannot get a GV number.**

- In the final step, choose a number and use a real mobile number for verification. Here I used a real US mobile number from SMS: https://sms-activate.guru/

- About keeping a GV number alive

Google Voice calls to `+1` numbers are free, but the number may be reclaimed if unused for around three months.
You can keep it active by making calls or sending texts from time to time.

```
1. Make or receive voice calls actively
Have two Google Voice numbers call each other and send texts to each other.

2. Call free phone numbers
ChatGPT phone version: +1(800)242-8478
Voice of America: +1(213)493-0288
Microsoft support: +1(800) 642-7676
Apple support: +1(800)275-2273
Amazon support: +1(206)266-2992

3. Subscribe to text messages
67135 send AUTO, reply Y, AutoZone 4 messages/mo
555444 send SIGNUP, reply Y, 6 msgs/mo
22122 send JOIN, Starbucks 2 msgs/mo
527365 send join, reply yes, 8 msgs/mo
25666 send code first, then reply PROMO
47272 send START, reply OK, 6 msgs/mo
22000 send info for an instant reply
22000 send help for a delayed reply
```

It is better to make calls directly. Texting is more likely to trigger risk control. Log in from a stable node in daily use.
